## 🎯 APERÇU DU PROJET

Ce projet a pour objectif d'analyser le marché immobilier sénégalais à partir de **4 313 annonces** collectées sur Expat-Dakar.com. Il se compose de :

- Une **API REST Django** servant les données
- Un **dashboard interactif React** avec visualisations graphiques
- Une **carte SVG interactive** du Sénégal
- Des **KPIs en temps réel** (prix moyen, médian, etc.)

---

## ✨ FONCTIONNALITÉS

### Backend (Django)
- ✅ API REST avec 8 endpoints
- ✅ Base de données SQLite (4 313 annonces)
- ✅ Documentation Swagger intégrée
- ✅ Filtres par ville, quartier, prix

### Frontend (React)
- ✅ Page d'accueil avec hero section et statistiques
- ✅ Dashboard avec 4 KPIs en temps réel
- ✅ Graphiques interactifs (Recharts)
  - Prix moyen par ville (bar chart)
  - Évolution mensuelle des prix (line chart)
  - Top quartiers les plus chers (tableau)
- ✅ Carte SVG du Sénégal interactive (survol = infos)
- ✅ Design responsive (mobile/tablette/desktop)
- ✅ Thème personnalisé (couleurs, ombres, transitions)
